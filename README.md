# HINDI-SIGN-LANGUAGE-TO-SPEECH-CONVERSION-USING-ARTIFICIAL-NEURAL-NETWORK-AND-MEDIAPIPE
Sign language recognition systems are designed to overcome the communication gap between hearing-impaired persons and the general public. Various sign language recognition systems have been proposed, employing sensor gloves, traditional image processing, and deep learning techniques. However, few systems are focused on Hindi/ISL sign language, and there is a need for a scalable solution in real-time environments. In this paper, a real-time sign language conversion from Hindi Sign Language to Speech is proposed, employing Media Pipe-based 21-hand landmark detection and a Convolutional Neural Network classifier. The detected landmarks are normalized and classified into Hindi characters using a neural network classifier. Temporal buffering logic is employed to generate words and sentences, and then convert them into speech format. The proposed system is hardware-independent, robust, and operates in real-time.
